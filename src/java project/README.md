(i) Project Title

Hotel Management System – Java + JDBC

(ii) Description

A simple console-based Hotel Management System built using Java and JDBC to manage room reservations. The project allows users to create, view, update and delete reservations while interacting directly with a MySQL database. It demonstrates clean modular code, proper database handling, error management, and real-world use of JDBC in backend development.
